# DeepRTE

Solve the following steady state radiative transfer equation

$$
\left \{
    \begin{aligned}
        & \mathbf{\Omega} \cdot \nabla \psi (\mathbf{r}, \mathbf{\Omega}) + \sigma_T \psi(\mathbf{r}, \mathbf{\Omega}) = \frac{1}{ |S| } \int \psi(\mathbf{r}, \mathbf{\Omega}') \, d\mathbf{\Omega}', \\
        & \psi_{\Gamma^{-}} = f.
    \end{aligned}
\right.
$$
